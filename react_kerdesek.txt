Milyen kiemelt tulajdonságai vannak egy React alkalmazásnak?

    - JSX : Javascript kiegészítve XML. HTML és javascript kódok egyvelege.
    - Componensek: újrafelhasználható UI elemek.
    - Virtual DOM: a böngészőben lévő oldal Memóriában leképzett megfelője.
    amin a React elsődlegesen a változtatásokat elvégzi és ezt szinkronizálja a "valós" DOM-al
    - One-way data binding: egyirányú adatkötést tesznek lehetővé
    - Nagy teljesítményű függvénykönyvtár

Mi a JSX?

    - HTML struktúra ami javascriptet tartalamzhat.
    <h1>
        <p>Bekezdés</p>
    </h1>

A böngésző tudja-e értelmezni a JSX-et?

    - NEM, mert a javascript kód szerves részét képezi, amit böngésző nem tud értelmezni.

Mi a virtual DOM?

    - DOM = Documnt Object Model. Ez a HTML dokumentumot reprezentálja.
    - Virtual DOM az a "valós" DOM-nak egy könnyű leképezése a valós DOM-nak.

Milyen újítások voltak az ES6-ban?

    - az osztályok (class) bevezetése
    - import bevezetése var React = require('react');
                         import React from 'react';
    - ()=> {}

Reactben a listák renderelése?

    const nevek = ["Robi","Feri","Anna"]

    const lista = ()=> {
        const listaElemek = nevek.map( (nev,_index) => {

        return <ul>{listaElemek}</ul>
